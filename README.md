## Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/Mangesh636/AuthMatrix.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Run the JSON server:
   ```bash
   npm run json-server
   ```
